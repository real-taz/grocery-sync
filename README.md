Attach a volume to the container in order to save data, Deploy the container
Default login admin/admin, port 3000.


1. Add your products: Name, Category, Barcode (you can scan with the camera), Units, Picture.
2. Create a new list & add items (You can scan with your camera the barcode to add items)
3. Enter the "Shopping Mode" to cross items off the list - again, you can use the camera to scan items barcode.


Limitation: Camera barcode scan supported on iPhone only if behind reverse proxy & https.
